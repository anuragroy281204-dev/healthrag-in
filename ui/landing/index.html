The Pipeline

From your question
to a cited answer.

01 — Retrieve

Hybrid Search

Your question is searched across 1,247 medical chunks using two retrieval methods in parallel. FAISS finds chunks with similar meaning via 384-dimensional embeddings. BM25 finds chunks with the exact terminology you used. Both result lists are fused via Reciprocal Rank Fusion to produce a single ranked top-5.

FAISS
BM25
RRF
02 — Ground

Strict Prompt

The retrieved chunks are packaged into a versioned system prompt with three non-negotiable rules. Use only these sources. Cite every factual claim with a numbered marker. Refuse explicitly if the sources cannot support the answer. The prompt also blocks personalized medical advice and detects emergencies.

Prompt v1.0.0
Refusal-aware
Safety guardrails
03 — Generate

Resilient Inference

Llama 3.3 70B (via Groq) writes the answer at temperature 0.1 for deterministic, source-faithful output. If Groq is rate-limited or unavailable, the system automatically falls back to Google Gemini 2.0 Flash. Same prompt, same output contract — only the underlying model changes.

Llama 3.3 70B
Gemini 2.0 Flash
Auto-failover
04 — Cite

Verifiable Output

Every factual claim ends with a [N] marker. Each marker maps to a real chunk with its source name, document title, and clickable URL. A citation validator checks that no marker points to a chunk that wasn't actually retrieved — catching hallucinated citations before they reach the user.

Differentiation

Built different,
on purpose.

◎

No hallucinations

The system literally cannot invent facts — it can only synthesize what was retrieved. Every claim must trace back to a chunk in the corpus or be refused.

⌖

Refusal-first

Out-of-scope and personal-advice questions are explicitly refused, not guessed at. Two layers: a code-level relevance gate, plus the prompt itself enforcing standardized refusal text.

▣

India-aware

ICMR guidelines and Indian PubMed studies are surfaced when relevant. HbA1c targets, dosing, and dietary recommendations reflect Indian population data, not Western defaults.

◇

Provider-resilient

Multi-provider LLM routing with automatic failover. Groq Llama 3.3 70B primary, Gemini 2.0 Flash secondary. Stays online when one provider rate-limits — production-grade redundancy.

⊞

Measured, not claimed

A five-metric LLM-as-judge evaluation harness scores every answer for faithfulness, relevance, context precision, citation accuracy, and refusal correctness. Every release has a published scorecard.

⌬

Open & free

Built entirely on free-tier APIs and open-source libraries. Source available on GitHub. Zero rupee operational cost, zero paywalled dependencies.
